# Dual Simplex Solver

A web-based linear programming solver that demonstrates the Dual Simplex Method with step-by-step visualization of the tableau transformations.

## Features

- Solve linear programming problems (minimization and maximization)
- Support for ≥, ≤, and = constraints
- Dynamic variable and constraint input (2-3 variables)
- Step-by-step tableau visualization with pivot highlighting
- Standard form conversion display
- Optimal solution with variable values and objective value
- Example problem loader for quick testing

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Building

```bash
npm run build
```
